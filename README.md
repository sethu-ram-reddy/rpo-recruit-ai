# AI Recruiting Copilot

An AI-powered recruiting dashboard designed to help recruiters manage candidates, resumes, and job pipelines efficiently.

## Overview

AI Recruiting Copilot is a modern recruiting platform that combines a clean recruiter dashboard with AI-driven candidate insights. The platform is designed to streamline hiring workflows by organizing candidates, resumes, and job postings into a single intelligent interface.

## Features

- Recruiter authentication system
- Dashboard overview of hiring pipeline
- Candidate management
- Resume tracking
- Job posting management
- Modern UI built for recruiting workflows

## Tech Stack

Frontend
- React
- Vite
- JavaScript
- CSS

Architecture
- Modular component-based frontend
- Service-based API layer
